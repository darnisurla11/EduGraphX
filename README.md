# 🎨 EduGraphX – AI Educational Infographic Generator

EduGraphX is an AI-powered infographic generation system that transforms complex topics into visually engaging, classroom-ready, and social-media-friendly educational infographics.

## 🚀 Features

* AI-driven topic analysis
* Visual knowledge maps
* Concept hierarchies
* Beginner-to-expert learning flow
* Educational infographic layouts
* Mobile and presentation-friendly outputs

## 📚 Key Computer Science Topics Covered

* Stack Operations Visualized
* Tree Traversal Techniques
* Binary Search Trees (BST)
* Hash Tables Simplified
* Data Algorithms
* Operating Systems


## 🎯 Applications

* Student learning
* Classroom teaching
* Exam preparation
* Research visualization
* Educational content creation

## 🛠️ Built With

* HTML
* CSS
* JavaScript
* Vite
* AI Prompt Engineering

## Run Locally

**Prerequisites:**  Node.js

## 🌐 Live Demo
➜  Local:   http://localhost:3000/
➜  Network: http://10.209.90.100:3000/


1. Install dependencies:
   `npm install`
2. Set the `GEMINI_API_KEY` in [.env.local](.env.local) to your Gemini API key
3. Run the app:
   `npm run dev`
